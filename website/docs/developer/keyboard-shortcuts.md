# Keyboard Shortcuts

## Timesheet

| Key | Action |
|---|---|
| `Insert` | Add new work entry |
| `F2` | Edit selected entry |
| `Delete` | Delete selected entry |
| `F5` | Refresh entry list |

---

## Dialogs (Add/Edit entry, Add client)

| Key | Action |
|---|---|
| `Enter` | Save / confirm |
| `Escape` | Cancel / close |
| `Tab` | Move to next field |
| `Shift+Tab` | Move to previous field |
